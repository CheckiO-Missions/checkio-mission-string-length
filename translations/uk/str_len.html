https://docs.python.org/3/library/functions.html. len це функція яку ти шукаєш.

Якщо ти відкриєш python консоль та впишеш "len('hi')" (без лапок ""), то ти отримаєш у відповідь 2, тому що у слові "hi" дві літери.

Добре. Як мені використати цю функцію для рішення задачі?

Твій код повивен виглядати так:

def string_length(text: str) -> int:
    # your code here
    return 0

всередні змінної "text" ти маєш першу змінну, котра string, і саме її двожину потрібно знайти. Тож, просто зроби наступне: 

len(text)
Останнє запитання. Як має виглядати мій фінальний код? 
def string_length(text: str) -> int:
    # your code here
    return len(text)
